HTML 基础结构

文档骨架、常用标签、语义化入门

HTML 是网页的骨架：先弄清文档结构（DOCTYPE、head、body），再记常用标签的语义与写法。期末卷里常考「哪个标签做什么用」「块级还是行内」这类概念题，编程题则要求能写出完整页面框架。

1.1 标准文档结构

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>页面标题（显示在浏览器标签栏）</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <!-- 可见内容写在这里 -->
</body>
</html>
标签/声明作用易错点
<!DOCTYPE html>声明 HTML5 文档类型，触发标准模式不是 HTML 标签，无闭合标签
<meta charset>声明字符编码，常用 UTF-8应放在 <head> 靠前位置
<title>浏览器标签页标题与 <h1> 不同，h1 是正文标题
<link rel="stylesheet">引入外部 CSS用 href 不是 src

1.2 文本与行内标签

标签含义块级/行内
h1~h6标题，h1 最大块级
p段落块级
br换行（空元素）行内/空元素，不是块级
hr水平分割线块级；粗细用 size 或 CSS
b / strong加粗（strong 语义更强）行内
i / em斜体（em 表强调）行内
sup / sub上标 / 下标行内

1.3 链接、图片与媒体

<a href="https://www.example.com" target="_blank">新窗口打开</a>
<img src="images/pic.jpg" alt="描述文字" width="300">
<video controls>
  <source src="video.mp4" type="video/mp4">
</video>
  • target="_blank"：新窗口/新标签页打开链接
  • alt：图片无法显示时的替代文字（必写，利于无障碍）
  • <source>：为 audio/video 提供多种格式，解决浏览器兼容

1.4 字符实体

空格&nbsp;
小于号 <&lt;
大于号 >&gt;
引号 "&quot;
版权 ©&copy;
和号 &&amp;

1.5 HTML5 语义化标签（了解）

标签用途
header页眉/区块头部
nav导航链接区域
article独立完整内容（新闻、评论、日志）
section文档中的主题分区
footer页脚

HTML 表格 ★

老师模拟卷重点：caption、rowspan、colspan

表格是填空题和编程题的高频考点。核心是 tr / th / td 的嵌套关系，以及 rowspan（跨行）、colspan（跨列）的数值含义。写表时先画草图再填标签，避免合并后行列对不齐。

2.1 基本结构

<table border="1" width="300px" cellspacing="0">
  <caption>学生信息表</caption>  <!-- 表格标题 -->
  <tr>
    <th>列标题</th>
    <td>单元格数据</td>
  </tr>
</table>
标签说明
table表格容器
caption表格标题（显示在表格上方）
tr表格行（table row）
th表头单元格（加粗居中，语义为标题）
td普通数据单元格
thead / tbody / tfoot表头区 / 主体 / 表脚（语义分组，了解）

2.2 单元格合并 ★★★

rowspan — 跨行（纵向）
<td rowspan="2">
向下合并 2 行
colspan — 跨列（横向）
<th colspan="3">
向右合并 3 列
记忆：row = 行 → rowspan 跨行；col = 列 → colspan 跨列。考试常考「学号 rowspan=2 + 个人信息 colspan=3」结构。

2.3 老师卷例题结构（学生信息表）

<table border="1" width="300px" cellspacing="0">
  <caption>学生信息表</caption>
  <tr>
    <th rowspan="2">学号</th>
    <th colspan="3">个人信息</th>
  </tr>
  <tr>
    <th align="center">姓名</th>
    <th align="center">性别</th>
    <th align="center">年龄</th>
  </tr>
  <tr>
    <td>007</td><td>李志玲</td><td>女</td><td>19</td>
  </tr>
</table>

2.4 常用表格属性

属性作用
border边框宽度（HTML 属性，建议用 CSS border 替代）
cellspacing单元格之间的间距
cellpadding单元格内边距
width表格/单元格宽度
align水平对齐：left / center / right

HTML 表单 ★

老师模拟卷重点：注册页、input 类型、label、required

表单负责收集用户输入。考试常考注册页、登录页的 HTML 结构：哪些控件用 input、如何用 label 关联、required 表示必填。注意区分「表单元素」与普通标签（如 img 不是表单控件）。

3.1 form 容器

<form action="/submit" method="post">
  <!-- 表单控件 -->
  <input type="submit" value="提交">
</form>
属性说明
action提交目标 URL
methodget（参数在 URL）或 post（在请求体）
enctype上传文件时必须 multipart/form-data

3.2 input 常用 type ★★

type用途示例
text单行文本用户名
password密码（掩码显示）密码框
email邮箱（带格式校验）注册邮箱
radio单选（同 name 为一组）性别
checkbox复选（可多选）兴趣爱好
file文件上传头像
submit提交按钮登录/注册
reset重置按钮清空表单
button普通按钮（配合 JS）确定

3.3 重要表单属性

属性作用考试频率
name提交时的字段名高
id唯一标识，配合 label / JS高
value默认值/按钮文字中
placeholder输入框灰色提示文字高
required必填，不填无法提交★ 常考
readonly只读，可提交中
disabled禁用，不可提交中
checked默认选中 radio/checkbox中
selectedoption 默认选中中

3.4 label 与控件关联

<label for="uname">用户名：</label>
<input type="text" id="uname" name="username">

for 的值必须等于 input 的 id，点击文字可聚焦输入框。

3.5 其他表单元素

标签说明
textarea多行文本，如自我介绍
select / option下拉选择框
fieldset / legend表单分组及分组标题（注册页常用）
button按钮，type="submit|reset|button"
易错：<img> 不是表单元素；required 是 HTML5 标准必填属性，不是 formvalidate / validate。

3.6 注册页 CSS 常考点（配合编程/填空）

  • 表单项标题右对齐：label { text-align: right; }
  • 输入框圆角：border-radius: 3px;
  • 焦点红色阴影：input:focus { box-shadow: 0 0 6px red; }

HTML 列表 ★

无序/有序/定义列表，导航与内容排版

列表用于导航菜单、步骤说明、术语解释等场景。三种列表不要混用：ul 无序号、ol 有序号、dl 是「术语 + 解释」的定义列表。导航栏通常用 ul > li > a 嵌套。

4.1 无序列表 ul

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

默认显示圆点项目符号，常用于导航、并列条目。

4.2 有序列表 ol

<ol>
  <li>打开浏览器</li>
  <li>输入网址</li>
  <li>按回车访问</li>
</ol>

默认显示数字序号，用于有先后顺序的步骤（如「十大理由」）。

4.3 定义列表 dl

<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>
  <dt>CSS</dt>
  <dd>层叠样式表</dd>
</dl>

dt = 术语/名称，dd = 解释说明。

4.4 列表 + 导航（老师卷名牌大学链接）

<ul>
  <li><a href="http://www.fudan.edu.cn">复旦大学</a></li>
  <li><a href="http://www.whu.edu.cn/">武汉大学</a></li>
</ul>
超链接放在 <li> 内；配合 CSS display:flex 可实现横向一行排列。

4.5 对比速记

类型标签子元素默认符号
无序ulli圆点 ●
有序olli数字 1. 2. 3.
定义dldt + dd无

CSS 弹性布局 Flex ★

一维布局核心，居中/导航/分栏必考

Flex 是现代 CSS 布局的基础，期末卷里选择题和填空题出现频率很高。记住三步：父元素设 display:flex，用 justify-content 管主轴对齐，用 align-items 管交叉轴对齐。导航栏水平排列、内容垂直居中都是典型考法。

5.1 启用 Flex

.container {
  display: flex;           /* 设为弹性容器 */
}

子元素自动成为 flex 项目（flex item），默认沿主轴排列。

5.2 主轴与交叉轴

概念说明
主轴 main axis项目排列方向，默认水平从左到右
交叉轴 cross axis与主轴垂直的方向
flex 容器设置了 display:flex 的父元素
flex 项目容器的直接子元素

5.3 容器属性 ★★★

属性常用值作用
flex-directionrow / column主轴方向：水平 / 竖直
justify-contentflex-start / center / flex-end / space-between主轴对齐
align-itemsflex-start / center / flex-end / stretch交叉轴对齐
flex-wrapnowrap / wrap是否换行
gap如 8px、12px项目间距
justify-content:flex-start = 主轴起始端对齐（水平时即左对齐）。align-items:center = 交叉轴居中（常做垂直居中）。

5.4 水平垂直居中模板 ★

.box {
  display: flex;
  justify-content: center;  /* 主轴居中 */
  align-items: center;      /* 交叉轴居中 */
  height: 200px;
}

5.5 导航横向排列模板

ul {
  list-style: none;
  padding: 0;
  display: flex;
  gap: 8px;
}
a {
  padding: 5px 10px;
  text-decoration: none;
}
a:hover {
  background: lawngreen;
}

5.6 两栏布局（专业宣传页思路）

.main {
  display: flex;
  gap: 20px;
}
.main-left {
  flex: 1;              /* 占据剩余空间 */
}
.notice {
  width: 360px;         /* 固定宽度侧栏 */
}

CSS 基础

引入方式、语法、单位

CSS 负责网页的外观与排版。先掌握三种引入方式（行内、内部、外部 link），再熟悉「选择器 + 声明块」的基本写法。单位上 px 最常用，百分比和 em 了解即可。

6.1 三种引入方式

方式写法优先级
内联style="color:red"最高
内部<style>...</style>中
外部<link rel="stylesheet" href="...">中

6.2 CSS 语法

选择器 {
  属性名: 属性值;
  属性名: 属性值;
}

6.3 常用单位

单位说明示例
px像素，绝对单位font-size: 16px
%相对父元素width: 50%
em相对当前元素 font-sizepadding: 1em
rem相对根元素 html 的 font-sizeline-height: 2rem
并非所有属性值都需要单位，如 opacity:0.5、line-height:1.5 可以无单位。

CSS 选择器与优先级

元素/类/ID/分组/后代 — 必考易混

选择器写法含义示例
元素p所有 p 元素p { color: red; }
类.类名class 匹配.p1 { }
ID#id名id 匹配（唯一）#main { }
并集p.p1同时满足（无空格）既是 p 又有 class p1
分组h1, p逗号 ORh1 和 p 都生效
后代p .p1有空格，p 内部的 .p1★ 易错
经典易错题：<p class="p1"> 用 p.p1 能选中，用 p .p1（有空格）不能选中。

优先级（从高到低）

  1. 内联样式 style=""
  2. ID 选择器 #id
  3. 类 / 属性 / 伪类 .class
  4. 元素 / 伪元素 p

CSS 盒模型

content → padding → border → margin

盒模型四层：内容 content → 内边距 padding → 边框 border → 外边距 margin。font 不属于盒模型。
属性方向写法记忆
paddingpadding-top/right/bottom/left内边距，内容与边框之间
marginmargin-left 等外边距，元素与元素之间
borderborder: 1px solid #ccc边框
width / height默认指 content 宽高配合 box-sizing 理解

box-sizing

CSS 动画与过渡

老师填空卷：@keyframes、animation、暂停

10.1 transition 过渡

适合简单属性变化。display 等离散属性不能过渡。

.box {
  transition: background-color 0.3s ease;
}

10.2 @keyframes + animation ★

@keyframes anim {
  0%   { transform: rotate(0deg); }
  50%  { transform: rotate(180deg); }
  100% { transform: rotate(360deg); }
}
div {
  border-radius: 50%;
  animation: anim 3s linear infinite;
}
div:hover {
  animation-play-state: paused;  /* 悬停暂停 */
}
属性含义
animation-name动画名称
animation-duration持续时间，如 3s
animation-iteration-count播放次数，infinite 无限
animation-play-statepaused 暂停 / running 播放

JavaScript 基础补充

只掌握期末常考内容，不做深入

JavaScript 在期末卷中占比不大，主要考基础语法和 DOM 常用方法。重点记 getElementById、事件属性（onclick、onsubmit、onkeydown）以及简单函数写法即可，不必深入 ES6 或框架。

11.1 变量与输出

console.log("姓名：" + name);   // 控制台输出
alert("总价：" + total);        // 弹框

11.2 函数

function compute() {
  var a = 5, b = 6;
  return a * b;
}
// 形参个数不限（考试选「任意」）

11.3 DOM 获取元素 ★

方法返回值
document.getElementById("id")单个元素对象
document.getElementsByTagName("p")元素集合

11.4 事件 ★

事件触发时机绑定对象

11.5 老师卷 JS 填空模板（价格计算）

var btn = document.getElementById('calc');
btn.onclick = compute;
function compute() {
  var price = document.getElementById('price');
  var number = document.getElementById('number');
  var total = parseInt(price.value) * parseInt(number.value);
  alert(total);
}
  • 按钮需设置 id="calc" 与 getElementById 对应
  • .value 获取输入框的值
  • parseInt() 转整数

11.6 运算符速记

运算符说明
===值和类型都相等（严格相等）
==只比较值，可能类型转换
+数字相加 / 字符串拼接
typeof返回类型字符串，如 "string"

考前速记清单

进考场前 10 分钟过一遍

这是进考场前的最后一遍扫盲：把容易混淆的概念再过一遍，尤其是表格合并、表单属性、Flex 对齐和 JS 对象方法。不必逐字背诵，但要能一眼认出正确选项。

HTML 必背

  • HTML 由 浏览器 解析执行
  • 注释：<!-- -->；DOCTYPE：<!DOCTYPE html>
  • 外部 CSS：<link rel="stylesheet" href="...">
  • 表格：rowspan 跨行，colspan 跨列；caption 标题
  • 表单：required 必填；placeholder 提示；label for = input id
  • 列表：ul 无序，ol 有序，dl 定义列表
  • br 不是块级元素；img 不是表单元素

CSS 必背

  • margin 外边距，padding 内边距
  • margin-left 不是 padding-left 的「左外边距」混淆题
  • flex：display:flex + justify-content + align-items
  • flex-start = 主轴起始端（左对齐）
  • position 默认 static
  • animation-iteration-count:3 播放 3 次
  • color 是文字色，不是背景色

JS 必背

  • getElementById → 单个对象
  • onsubmit → Form 对象
  • onkeydown → 按键盘
  • createElement → 创建元素
